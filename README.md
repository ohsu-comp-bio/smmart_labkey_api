# SMMART Labkey API

## Installation

See docs at: https://github.com/LabKey/labkey-api-python

Install with `pip`:

```
$ pip install labkey
```

And set up netrc file at ``~/.netrc``:
```
machine <remote-instance-of-labkey-server>
login <user-email>
password <user-password>
```
